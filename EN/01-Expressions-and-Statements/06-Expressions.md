# Expressions
Sequences of operators, literals and variables which represent a value
* Consist of at least one **operand**
* Can have 1 or more **operators**
```java
int y = x + 5;
```
```java
String name = "John Doe"
```
```java
int r = (150-20) / 2 + 5;
```
