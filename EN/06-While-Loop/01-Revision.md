# Prefix and Postfix
Prefix decrement
```java
int a = 1; 
System.out.println(--a); // 0
System.out.println(a);   // 0
```

Postfix decrement
```java
int a = 1;
System.out.println(a--);
System.out.println(a);
```

# Simple For-Loop
For loops repeat a certain code block a known number of times

Printing the numbers from 1 to 10
```java
for (int i = 1; i <= 10; i++) {
  System.out.println(i);
}
```
